3) Background System (MOST IMPORTANT)

Your screenshots show a background that is not plain white. It is:

a smooth gradient

with subtle noise (optional)

with blur depth

3.1 Base Background

Use a full-page background with:

gradient blobs

slight blur

Implementation spec:

A div behind everything:

fixed position

inset-0

z-index: -1

pointer-events none

3.2 Gradient Style (Reference Match)

Use 2–3 radial gradients:

Top left blob: purple/blue

Center blob: pink/violet

Bottom right: teal/blue

Opacity low — should not distract.

3.3 Optional: Noise Overlay

Add subtle noise texture:

opacity: 0.05 to 0.12

blend mode: overlay

This makes it premium.

4) Typography System
4.1 Font Choice

To match modern minimal UI, choose:

✅ Inter (recommended)
OR
✅ Satoshi (more stylish)
OR
✅ Plus Jakarta Sans (clean premium)

(Inter is safest.)

4.2 Typography Scale

Use clear hierarchy:

Hero headline (Home)

font-size: 42–56px (desktop)

font-size: 26–32px

7) Animation & Motion Design
7.1 Global Motion Rules

everything transitions smoothly

keep durations short

avoid “bouncy” motion

Timing

page transitions: 300–450ms

easing: ease-out / cubic-bezier(0.2,0.8,0.2,1)

Animations required

Button hover lift

page fade/slide transitions

splash cursor effect (from your code)
